# Time API

Dummy API to show time and to show setup of node based apps.

Full api reference is available under /api-docs path

## Build and run

- Install dependencies
```
npm install
```

- Dev run
```
npm start
```

- Prod build
```
npm run build
```
- Prod run
```
npm run dist/index.js
```

- Docker build
```
docker build -t abondar/timeapi .
```
